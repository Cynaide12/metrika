# kst_golang_server_template
Шаблон для быстрого начала разработки бэкенда на Go
## Описание структуры:
- **cmd/main.go** - инициализация приложения
- **config** - .yaml файл для конфига
- **internal/config** - парсинг .yaml файла в структуру конфига
- **internal/http-server** - тут все роуты и все что касается обработки http запросов
- **internal/logger** - логирование(с ротацией)
- **internal/models** - модели для работы с бд
- **internal/pkg** - доп. пакеты, которые используются только в этом бэкенде и не переходят в другие
- **internal/services** - внутренняя логика, просто слой сервиса
- **internal/storage** - все функции работы с БД, слой репозитория
- **lib** - мелкие функции которые могут переходить из проекта в проект
